# DevOverflow

This Next.js 14 project provides a robust question and answer platform akin StackOverflow.

## Project Overview

The project aims to create a space where users can ask and answer questions, collaborate on knowledge, and discover valuable information. It is built with a focus on user experience, community engagement, and efficient knowledge sharing.

## Features

- **Authentication:** Secure user sign-in and sign-up using Clerk.
- **Question Creation:** Users can ask questions with rich text formatting and tagging.
- **Answer Submission:** Users can provide detailed answers to questions, utilizing markdown for formatting.
- **Vote System:** Upvote or downvote questions and answers to signal quality.
- **Community:** Browse questions, explore tags, and interact with other users.
- **Profile Management:** Users can edit their profiles and track their contributions.
- **Search:** Quickly find relevant questions and answers using a powerful search engine.

## Installation

1. **Clone the repository:** 
   ```bash
   git clone https://github.com/Jakex-123/devflow
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
   
## Usage

1. **Start the development server:**
   ```bash
   npm run dev
   ```
2. **Access the application:** 
   Open `http://localhost:3000` in your browser.
